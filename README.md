5. Smart Streak Protection
   To keep users coming back, the "Streak" is the most powerful psychological tool.

The Feature: "Streak Freeze" items.

The Economy: Users can spend 500 LP to buy a "Protector." If they miss a day, their streak doesn't reset to zero.

Learning is more fun when it's competitive.

The Feature: A "Duel" mode. A user can send a challenge link to a friend. Both take the same 60-second rapid-fire quiz.

The Reward: The winner takes a small amount of "LP" (Learning Points) from the loser, or earns a "Warrior" badge.
